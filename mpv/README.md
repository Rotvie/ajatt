# Lua Scripts

Scripts included in this folder (`scripts/`):
- [autoload](https://github.com/mpv-player/mpv/blob/master/TOOLS/lua/autoload.lua): Automatically loads the other files in a directory into the playlist
- [mpv-bookmarker](https://github.com/NurioHin/mpv-bookmarker): Saves video bookmarks to a list accessible from within mpv (`bookmarker-menu.lua`)
- [recent](https://github.com/hacel/recent): Keeps a history of every file played in mpv
- [thumbfast](https://github.com/po5/thumbfast): High-performance on-the-fly thumbnails for the seekbar
- [uosc](https://github.com/tomasklaen/uosc): Feature-rich minimalist proximity-based UI
- `play-on-startup.lua`: Small custom snippet that unpauses playback as soon as a file loads

Installed separately (not in this repo):
- [mpvacious](https://github.com/Ajatt-Tools/mpvacious): Automatically creates Anki cards from mpv — the core of the sentence-mining workflow
